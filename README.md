# Prism

## 개발 시작

```bash
# 프론트엔드 (Vite + React + TypeScript + Tailwind)
cd frontend
npm install
npm run dev      # http://localhost:5173

# 백엔드 (Node.js + Express + TypeScript)
cd backend
npm install
npm run dev       # http://localhost:3001
```

## 구조

- `frontend/` — Vite + React + TypeScript + Tailwind CSS
- `backend/` — Node.js + Express + TypeScript (`GET /api/health` 헬스체크 제공)
